Ashreigns Empire discord bot is officially made by ashreignsmc ( by me ) for my own discord server.

This bot have features like:

- Listing avaliable commands
- Ban, kick, mute and warn commands
- announce #channel msg Post announcement
- userinfo [@user] User details
- clear [n] Delete messages
- hello Greet the user
- ping Latency check
- serverinfo Server details
- poll Yes/no poll
- restart Owner-only restart
